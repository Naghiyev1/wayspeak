# WaySpeak v1.5

## What changed

- Added Arabic phrasebook
- Added Persian phrasebook
- Added English ↔ Arabic presets
- Added English ↔ Persian presets
- Added Arabic romanization
- Added Persian romanization
- Added Persian to the language selector
- Romanization only appears for Chinese, Japanese, Korean, Arabic and Persian
- Arabic/Persian output displays right-to-left
- Speak button uses ar-SA for Arabic and fa-IR for Persian where supported
- Updated cache to v1.5

## Notes

Phrasebook romanization is pre-written and more reliable.

Live/API romanization is a rough fallback. Arabic and Persian transliteration is especially approximate because short vowels are often not written in the script.
